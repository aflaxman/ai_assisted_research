🛠 Settings & OneBusAway API Key

This dashboard ships with the public TEST key for OneBusAway Puget Sound. It works but is rate-limited and shared with everyone. For reliable use, get your own free key:

  1. Visit Sound Transit Open Transit Data and follow the OneBusAway API instructions.
  2. Or email api@onebusaway.org to request a key (free, usually replies within a day).
  3. Paste your key below and hit Save. It's stored in your browser only (localStorage), never sent anywhere except OBA.
